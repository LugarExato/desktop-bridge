# Lugar Exato Bridge

Este repositório é utilizado exclusivamente para hospedar as versões de distribuição e atualizações automáticas (Motor OTA) do Lugar Exato Bridge.

Nenhum código-fonte sensível está armazenado neste repositório.

Para baixar a versão mais recente, acesse a aba lateral "Releases" e efetue o download do arquivo de Setup ou Portable.
